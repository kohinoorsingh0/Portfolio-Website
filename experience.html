The Journey

How I got here.

Early 2025

Started BCA & First Lines of Code

Began my Bachelor's in Computer Applications and wrote my first HTML and CSS pages. Fell in love with the idea that a browser could turn text into something visual.

Mid 2025

JavaScript & Interactive UIs

Moved from static pages to interactive ones. Built calculators, to-do lists, and small games to understand the DOM and core JavaScript logic.

Late 2025

First React Components

Picked up React and started thinking in components and state instead of one long script. Rebuilt earlier vanilla-JS projects to compare approaches.

Early 2026

Node.js, Express & APIs

Went past the front end and learned to build REST APIs with Node and Express, connecting them to real interfaces for the first time.

Mid 2026

MongoDB & Full-Stack Projects

Added MongoDB to the stack and shipped my first complete MERN applications, handling everything from schema design to the final UI.

2026 — Now

Refining & Building This Portfolio

Focused on writing cleaner code, learning authentication and deployment, and polishing personal projects like this portfolio into something I'm proud to share.
